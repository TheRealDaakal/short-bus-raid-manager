# 🚍 Short Bus Raid Manager

A Discord bot for organizing **Star Wars: The Old Republic** Operations and raid signups.

## ✨ Features

- 🎲 Random Operation Wheel
- 🛡️ Tank signup
- 💚 Healer signup
- ⚔️ DPS signup
- 🪑 Bench system
- ❌ Leave raid
- 🔄 Automatic role limits

## 🛠️ Built With

- Python 3.14
- discord.py
- Git
- GitHub
