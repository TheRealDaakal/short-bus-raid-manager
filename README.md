# 🚍 Short Bus Raid Manager

A Discord bot for organizing SWTOR Operations.

## Features

- 🎲 Random Operation Wheel
- 🛡 Tank signup
- 💚 Healer signup
- ⚔ DPS signup
- 🪑 Bench system
- ❌ Leave raid
- Automatic role limits
